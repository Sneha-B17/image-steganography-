# Image Steganography System 🔐🖼️

## 📌 Description
This project hides secret data inside an image using bit-level manipulation in C.

## ⚙️ Features
- Encode secret message into image
- Decode message from image
- Uses LSB (Least Significant Bit) technique

## 🛠️ Technologies Used
- C Programming
- File Handling
- Bit Manipulation

## ▶️ How to Run
gcc main.c encode.c decode.c -o steg
./steg

## 📂 Sample Files
- input_image.bmp
- output_image.bmp
- message.txt

## 💡 Learning Outcome
Learned how data hiding works and improved low-level programming skills.
